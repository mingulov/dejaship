# Agent Simulation Tools

This directory is reserved for helper commands that should not run during normal pytest execution.

Planned commands:

- `generate_llm_fixtures.py`
- `validate_llm_fixtures.py`
- `summarize_llm_fixtures.py`
- `run_agent_sim_report.py`
- `evaluate_cross_model_retrieval.py`
- `evaluate_similarity_thresholds.py`
- `evaluate_embedding_ablation.py`
- `run_quality_suite.py`
- `compare_quality_bundles.py`

These commands will use the typed contracts in `_support/` and the fixture files in `fixtures/`.
